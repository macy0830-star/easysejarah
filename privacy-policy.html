Dasar Privasi

Tarikh Kuat Kuasa: 13 Julai 2025

EasySejarah.com ("kami") komited untuk melindungi privasi pengguna laman web ini. Dasar Privasi ini menerangkan bagaimana kami mengumpul, menggunakan, dan melindungi maklumat anda.

1. Maklumat Yang Kami Kumpul

Kami mengumpul maklumat tanpa nama (anonymous) melalui Google Analytics 4 (GA4) termasuk: jenis peranti, pelayar web, lokasi umum (negeri/negara), halaman yang dilawati, dan tempoh masa di laman web. Kami TIDAK mengumpul nama, e-mel, atau maklumat peribadi yang boleh mengenal pasti anda.

2. Penggunaan Kuki (Cookies)

Laman web ini menggunakan kuki untuk tujuan analitik melalui Google Analytics. Kuki adalah fail teks kecil yang disimpan dalam peranti anda. Anda boleh mematikan kuki melalui tetapan pelayar anda, namun sesetengah fungsi laman web mungkin terjejas.

3. Google Analytics

Kami menggunakan Google Analytics 4 untuk memahami bagaimana pengguna berinteraksi dengan laman web kami. Google Analytics mengumpul data secara tanpa nama. Untuk maklumat lanjut tentang cara Google menggunakan data ini, sila lawati: policies.google.com/privacy.

4. Pengiklanan (Google AdSense)

Laman web ini mungkin menggunakan Google AdSense untuk memaparkan iklan. Google AdSense menggunakan kuki untuk memaparkan iklan berdasarkan lawatan terdahulu anda ke laman web ini dan laman web lain. Anda boleh menyahpilih penggunaan kuki peribadi dengan melawati Tetapan Iklan Google.

5. Pautan Pihak Ketiga

Laman web ini mungkin mengandungi pautan ke laman web pihak ketiga seperti YouTube dan Shopee. Kami tidak bertanggungjawab ke atas dasar privasi laman web tersebut.

6. Perubahan Dasar Privasi

Kami berhak mengubah Dasar Privasi ini pada bila-bila masa. Perubahan akan berkuat kuasa serta-merta selepas disiarkan di laman web ini.

7. Hubungi Kami

Sekiranya anda mempunyai sebarang pertanyaan berkaitan Dasar Privasi ini, sila hubungi kami melalui e-mel di hello@easysejarah.com.
